# Mezzo Task Timer

[![Mezzo Task Timer](https://mezzotimer.com/images/mezzo.png)](https://mezzotimer.com)

### To Build:  

`npm install`  
`npm run dist-mac` or `npm run dist-win`  

### To Develop:
  
`npm run start-mac` or  `npm run start-win` 

### Built using:   
node 16.0.0  
electron 13.1.2

### To Download Prebuilt Release:
